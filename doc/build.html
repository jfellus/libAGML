libAGML User's Documentation

Prerequisites

The following packages are required for building and running:

  • C++ compilations tools (g++ compilation tools, version 4.7 or higher is recommended)

  • CMake, version 2.6 or higher

  • Dynamically Loaded (dl) and POSIX.1b Realtime Extensions (rt) libraries (already installed in most Linux distributions)

Configure and build

In the project directory, run the following commands:

$ ./configure
$ make

The configure script will create a build dir and run cmake; The make command will call the Makefile created by cmake in the build dir.

Run

All generated binaries are created in the bin directory.

To run one example, go to the example/ directory, and then run one of the .sh files, for instance:

$ cd examples
$ ./kmeans_single_host.sh
